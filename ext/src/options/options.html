The Filter UI Component

Disable/Enable Filter UI Element

The extension adds the filter element (shown above) to let you apply filters to the shows on release calendar. If you have a filter that you always use, you can lock the filter and then disable the UI. The filters (as long as it is locked) will still filter shows even if the UI is hidden.

Quick Select Dub Languages

The "Shown Languages" box contains the languages available for quick switching on the filter UI element. The available languages are listed in the "Available Languages" box. Use the "Add" and "Remove" buttons to move languages between the boxes. In the filter element (shown above) the options under "Enable Dubs for" let you quickly pick the languages of the dubs you want to see or hide. The Hidden Count UI Component

Disable/Enable Hidden Count Reflow

The extension preforms a reflow in order to show you the orange flash whenever the hidden count changes (shown above) after the first time. The number of shows hidden will still update even if this is disabled.

Disable/Enable Hidden Count UI Element

The extension adds the '# Hidden' element (shown above) to convey how many shows are hidden when you apply filters. The filters will still hide shows even if this is disabled.
